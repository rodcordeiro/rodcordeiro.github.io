Oncopy function

Hoje eu tava lendo algumas funções tipo onclick e oncopy, que atuam no momento em que a ação bem intuitiva é iniciada, por exemplo a onclick quando o botão ou link é clicado e a oncopy quando o texto é copiado.

<input type="text" oncopy="myFunction()" value="Try to copy this text">
<input type="text" oncut="myFunction()" value="Try to cut this text">
<p> contenteditable="true" onpaste="myFunction()">Try to paste something inside this paragraph</p>

Digievolução

DOM, 22/04/18

Hoje eu comecei a atualizar o blog. Coloquei os primeiros cards e formatei eles. Estou gostando deste projeto, esta bem interessante. O lado negativo é que acabo passando ainda mais tempo voltado ao celular, preciso aprender a conciliar isso.

Uma coisa que eu aprendi hoje é a bloquear a seleção do texto, ou seja, não dá para selecionar o texto, ou deixar para selecionar apenas o parágrafo inteiro. para isso basta colocar o código abaixo no css:

content_copy
-webkit-user-select:none; /* Safari 3.1+ */
-moz-user-select: none;/* Firefox 2+ */
-ms-user-select: none;/* IE 10+ */
user-select: none; /* Standard syntax */

A sintaxe deste atributo é:

user-select: auto | none | text | all;
